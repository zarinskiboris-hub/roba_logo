ROBA Roadmap

A clear path from the tokenized dev marketplace to full-stack robotics and cloud simulation.

View Timeline Earn ROBA
Phase 1 MVP

Foundation & Tokenized Tasks

Auth, roles, GitHub import, task marketplace, ROBA rewards.

  • JWT + OAuth (GitHub/Google)
  • Role-based access (Dev/Enterprise/Admin)
  • Task marketplace & leaderboards
  • ROBA rewards & wallet connect
  • Asset versioning (core moat)
Phase 2 Simulation

Cloud Simulation (Docker + ROS2 + Gazebo)

Containerized robotics jobs with live telemetry and recordings.

  • Docker GPU images (ROS2 Humble, Gazebo)
  • Job queue, status, artifact retrieval
  • WebSocket telemetry & video capture
  • Benchmark harness & leaderboards
  • Kitchen/warehouse demo templates
Phase 3 Code Exec

Classic Code Execution (Internal Server)

Secure, sandboxed execution for common stacks with on-chain proofs.

  • Ephemeral sandboxes per run (Firecracker/Docker)
  • Supported stacks: Vue, Nuxt, React, Next.js, Node.js, Python
  • Build/test pipelines & artifact previews
  • Signed run attestations & reward distribution
  • Usage quotas, rate limits, and cost controls
Phase 4 Robotics

ROBA Studio for Robotics

Full robotics IDE-in-the-cloud with templates, marketplace, and governance.

  • Studio UX over Gazebo: scene graph, sensors, URDF loader
  • Template marketplace (Nav2, SLAM, vision, manipulation)
  • Enterprise governance (audit trails, SSO, compliance)
  • On-chain licensing & payouts for creators
  • Multi-robot orchestration & telemetry analytics

Timeline

  • On-chain distribution of $ROBA to contributors
  • Creator royalties for template marketplace
Integrations

  • Container registry, S3-compatible storage
